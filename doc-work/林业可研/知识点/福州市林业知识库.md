# 林业可行性研究报告知识点库

## 福建省福州市林业特点

### 地理环境
- 亚热带季风气候
- 年均气温19-21°C
- 年降雨量1200-1800mm
- 森林覆盖率约65%（全国前列）

### 林业类型
- 用材林：杉木、马尾松、竹林
- 经济林：茶果、油茶、板栗
- 生态林：阔叶林、混交林
- 薪炭林：相思树、木麻黄

## 林业专业术语

| 术语 | 解释 |
|------|------|
| 郁闭度 | 森林冠层投影面积与林地面积之比 |
| 蓄积量 | 单位面积森林中林木的材积总量 |
| 公顷 | 1公顷=15亩=10000平方米 |
| 轮伐期 | 两次采伐之间的间隔时间 |
| 乔木 | 树干高大、树冠茂密的树木 |
| 灌木 | 矮小的木本植物 |
| 造林成活率 | 造林后成活株数/造林株数×100% |
