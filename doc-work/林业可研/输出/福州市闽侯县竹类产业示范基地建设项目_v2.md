# 福州市闽侯县竹类产业示范基地建设项目

## 第一章 项目总论

### 1.1 项目概况
- 项目名称：福州市闽侯县竹类产业示范基地建设项目
- 项目地点：福州市闽侯县白沙镇
- 建设规模：500公顷（7500亩）
- 总投资：850万元

### 1.2 项目建设背景
闽侯县位于福州市西南部，属亚热带季风气候，年均气温19.5°C，年降雨量1450mm，适合竹类生长。全县现有竹林约8万亩，但集约经营程度不高。

福州市"十四五"林业发展规划提出"做大做强竹产业"，《闽侯县林业产业发展规划》将竹产业列为支柱产业。

### 1.3 项目建设必要性
1. 政策契合：符合省市县三级林业规划
2. 资源优势：气候适宜，劳动力充足
3. 效益显著：竹类3年成林，年年有产出
4. 带动明显：促进农民增收致富

### 1.4 项目主要结论
本项目选址合理，技术可行，效益显著，建议组织实施。
