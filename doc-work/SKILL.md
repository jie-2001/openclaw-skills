# 文档工作处理 Skill

## 概述
私人管家文档处理系统，支持Word扩写、文件仿写、PPT/PDF处理

## 文件结构
```
doc-work/
├── SKILL.md           # 本文件
├── 林业可研/          # 林业可行性研究报告
│   ├── 模板/          
│   ├── 参考资料/
│   └── 输出/
├── 工程类/            # 工程项目文档
└── 经济类/            # 经济分析文档
```

## 功能列表
1. Word文件扩写
2. 文件仿写
3. PPT创建/仿做
4. PDF处理

## 触发条件
当用户需要处理文档时激活
