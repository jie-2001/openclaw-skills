# langflow Skill

## 概述
AI工作流可视化工具 - Langflow (144k⭐)

Langflow是一个强大的AI应用构建和部署工具，支持可视化编排AI工作流。

## 项目信息
- **Stars**: 144k+
- **GitHub**: https://github.com/langflow-ai/langflow
- **官网**: https://langflow.org

## 功能
- 可视化AI工作流编排
- 支持多种LLM集成
- RAG管道构建
- Agent工作流设计

## 本地部署
```bash
# Docker部署
docker pull langflowai/langflow
docker run -d -p 7860:7860 langflowai/langflow

# 或 pip安装
pip install langflow
langflow
```
